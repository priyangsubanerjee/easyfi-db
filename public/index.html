API Documentation.

Developers can upload files (one file at a time) by sending POST request.

Status: . Operational

1Upload a file

Send a POST request to https://easyfidb.herokuapp.com endpoint with your form data.

Request

const file = e.target.files[0];
const formData = new FormData();
formData.append('userImage', file);
axios.post('https://easyfidb.herokuapp.com/upload-file', formData)
    .then(res => {
        console.log(res)
})

Response

{
    "fileUrl": "http://easyfidb.herokuapp.com/uploads/9a0c020a-6b2f-4460-8dde-d42e8a5e1597.png",
    "fileName": "diamond_logo.png",
    "fileType": "image/png",
    "fileSize": "28.74kb"
}

2Delete a file

Request

const fileName = "ascccridoe-deds.jpg";
axios.post('http://easyfidb.herokuapp.com', JSON.stringify({name: fileName}))
    .then(res => {
        console.log(res)
})

Response

{ "deleted": true }
